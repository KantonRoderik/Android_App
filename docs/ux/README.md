## Áttekintés
Az alkalmazás egy modern, sötét tónusú és zöld/kék akcentusokkal rendelkező fitnesz követő, amely segít a felhasználóknak a napi kalória- és vízfogyasztásuk nyomon követésében. Támogatja a manuális rögzítést, a vonalkódos keresést és az automatikus célmeghatározást is.
