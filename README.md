# SZAKDOLGOZAT

## Quickstart
1. Clone the repository.
2. Open in Android Studio.
3. Sync Gradle.
4. Run on an emulator or physical device.

## Test Run
- Unit tests: `./gradlew test`
- Android tests: `./gradlew connectedAndroidTest`

## Configuration
- Copy `.env.example` to `.env` (if applicable for backend/scripts).
- Ensure `google-services.json` is present in the `app/` directory for Firebase features.
